## Задача: Создать атомарный план задач

### Что нужно сделать:
Прочитай задачу от пользователя и разбей ее на логические шаги (подзадачи).

### Что создать:

**0. Очистка старых задач (выполни в терминале):**
```bash
rm -rf .tasks/todo/* .tasks/done/* 2>/dev/null || true
mkdir -p .tasks/todo .tasks/done
```

**1. Индекс** `.tasks/plan.md`:
```markdown
# План
Создан: [дата]

## Очередь
- [ ] task-001 — название (03-fix-simple-bug)
- [ ] task-002 — название (05-refactor)

## Выполнено
(пусто)
```

**2. Файл на каждую задачу** `.tasks/todo/task-NNN.md`:
```markdown
# Задача NNN: [название]

## Проблема
[описание]

## Файлы
`src/components/MyComponent.tsx`
`src/api/types.ts`
(Укажи только нужные файлы, без номеров строк)

## Решение
[конкретное решение]

## Промпт
[ТОЛЬКО из списка ниже]

## Тестирование
[TESTS_REQUIRED: true/false] (укажи true только если для этой задачи критически важны тесты, по умолчанию false)
---
```

### ВАЖНО: допустимые значения поля "Промпт"
Используй ТОЛЬКО эти значения:
| Промпт | Когда использовать |
|--------|--------------------|
| `03-fix-simple-bug.txt` | баг: лишний ререндер, неправильный стейт, неверная обработка API |
| `04-create-architecture.txt` | создание новых модулей, страниц или сложных компонентов с нуля |
| `05-refactor.txt` | простой рефакторинг: вынос логики в хук, разбиение компонента |
| `05b-refactor-complex.txt` | сложный рефакторинг: смена стейт-менеджера, переработка системы слоев |
| `06-write-unit-tests.txt` | написание тестов (только если TESTS_REQUIRED: true) |
| `08-lint-and-fix.txt` | проверка линтера и TS (добавляй ПОСЛЕДНЕЙ задачей, если пользователь указал LINT_REQUIRED: true или явно попросил проверить код) |
| `09-update-readme.txt` | обновление документации по итогам всех задач |
| `10-create-ui-component.txt` | верстка UI-компонентов по макету/фигме (без интеграции API и сложной логики, только dumb-компоненты) |
| `11-integrate-logic.txt` | подключение API и стейта к уже готовым UI-компонентам (Smart components, маппинг данных без изменения верстки) |

НЕ придумывай новые имена промптов.

После создания файлов: "✅ План готов: Задач: X"
