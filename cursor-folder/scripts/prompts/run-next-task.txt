## Выполни следующую задачу из плана

### Шаг 1: Найди следующую задачу
```bash
grep -m 1 '^\- \[ \]' .tasks/plan.md
```
Эта команда вернёт первую незавершённую строку, например:
`- [ ] task-001 — название (03-fix-simple-bug)`

Извлеки номер задачи из этой строки: `task-001`

### Шаг 2: Прочитай файл задачи
```bash
cat .tasks/todo/task-001.md
```
(используй реальный номер из шага 1)

### Шаг 3: Прочитай промпт задачи
Возьми значение поля "Промпт" из файла задачи и прочитай его:
```bash
cat .cursor/scripts/prompts/03-fix-simple-bug.txt
```
(используй реальное имя промпта из поля "Промпт" в задаче)

### Шаг 4: Прочитай целевые файлы
Прочитай файлы, указанные в блоке `## Файлы` в файле задачи.
```bash
cat src/components/MyComponent.tsx
```

### Шаг 5: Выполни задачу
Следуй инструкциям из промпта (шаг 3).
Используй детали из файла задачи (шаг 2): проблема, решение.
Пиши тесты, ТОЛЬКО если в файле задачи стоит `[TESTS_REQUIRED: true]`. Иначе — без тестов.

### Шаг 6: Закрой задачу
Перемести файл задачи в выполненные:
```bash
mkdir -p .tasks/done
mv .tasks/todo/task-001.md .tasks/done/task-001.md
```
(используй реальный номер)

Обнови `.tasks/plan.md` — перенеси строку из "Очередь" в "Выполнено", поставь ✅ и дату выполнения.

---
**КРИТИЧЕСКОЕ ПРАВИЛО:** Твой финальный ответ пользователю должен состоять РОВНО ИЗ ОДНОЙ СТРОКИ:
"✅ task-NNN завершена: [кратко что сделано]. Следующая: task-NNN — [название]"

**ЗАПРЕЩЕНО** писать "Я нашел задачу", "Сейчас я изменю файл", "Вот что я сделал" или использовать русский язык для болтовни. Только код и одна финальная строка.
**ЗАПРЕЩЕНО** начинать следующую задачу! Остановись и жди, пока пользователь запустит новый чат.
